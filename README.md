## 🛡️ 天锐绿盾 (TopGreen Shield) - 守护企业核心数据

### 为什么选择天锐绿盾？

✅ **透明加密技术** - 无感知加密，不影响员工工作效率  
✅ **多场景覆盖** - 支持CAD、Office、代码、设计文件等全格式保护  
✅ **智能权限管理** - 细粒度控制文件外发、复制、截屏、打印  
✅ **跨平台支持** - Windows、Linux、macOS 全平台客户端  
✅ **开发者友好** - 提供API接口，支持与企业现有系统无缝集成  

### 🔧 技术亮点

| 特性 | 描述 |
|------|------|
| 实时加解密 | 内核级驱动，毫秒级响应 |
| 离线策略 | 支持离线办公，策略本地缓存 |
| 审计追踪 | 全生命周期操作日志，可追溯 |
| 外发审批 | 灵活的外发审批流程，支持移动端 |
| 代码防泄漏 | 针对Git/SVN等版本控制特殊优化 |

### 💼 适用场景

- 🏢 **研发型企业** - 源代码、算法、架构图保护
- 🏭 **制造业** - CAD图纸、工艺文件、BOM表加密
- 🏦 **金融行业** - 客户资料、交易数据、报表防泄密
- 🏥 **医疗健康** - 患者隐私、研发数据合规保护

企业级数据防泄漏解决方案 - 透明加密、权限管控、审计追踪
